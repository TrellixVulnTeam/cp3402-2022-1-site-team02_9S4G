    Local Environments

    Local environments consisted of a Bitnami package coupled with Wordpress to host the local web server and database. For theme editing PhpStorm was the IDE of choice assisted by Sass and Grunt.js for coding CSS files and compiling, respectively. Commands to push the theme files to DEV and PROD sites were done using SSH tool of choice (Putty, for instance).
